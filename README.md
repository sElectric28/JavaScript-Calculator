# JavaScript-Calculator

![Screenshot here](/calculatorScreenshot.png)

## How to use:
1. Press any number button (e.g., 5)
2. Press an math operator button (e.g., *)
3. Press any number button (e.g., 8)
4. Press the equals button ('=')
5. The result should be diplayed in the input field (e.g., 40)
6. Press the 'AC' button to clear all data.
7. Press the '%' to decimate the number shown in the display

## How does it work
1. The first number is stored in an array
2. The operator value is saved in a variable
3. The second number is stored in an array
4. The first number is calculated with the second number
5. The result is shown in the display input value
